Simple virtual keyboard inspired by mobile keyboards.

[Documentação](https://eduhds.github.io/teclado.js)

## Instalação

```sh
npm i eduhds/teclado.js
yarn add eduhds/teclado.js
pnpm add eduhds/teclado.js
```

## Desenvolvimento

```sh
# Limpar outputs
pnpm run clean

# Compilar lib
pnpm run tsc

# Executar testes
pnpm run test

# Gerar documentação
pnpm run docs
```
